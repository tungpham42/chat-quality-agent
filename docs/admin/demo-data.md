# Dữ liệu demo

CQATP cung cấp bộ dữ liệu demo gồm **220 cuộc hội thoại mẫu** từ kịch bản quán coffee. Giúp bạn trải nghiệm đầy đủ tính năng trước khi kết nối kênh chat thật.

## Import dữ liệu demo

### Từ Dashboard

Khi công ty chưa có dữ liệu, Dashboard sẽ hiển thị banner:

> "Bạn chưa có dữ liệu. Import dữ liệu demo để trải nghiệm?"

Bấm **Import dữ liệu demo**.

### Dữ liệu demo bao gồm

- **220 cuộc hội thoại** mẫu (đa dạng kịch bản: khiếu nại, hỏi giá, đặt bàn, góp ý...)
- **Kênh chat giả lập** để chứa dữ liệu
- Tin nhắn có nội dung thực tế bằng tiếng Việt

Sau khi import, bạn có thể:

1. Xem tin nhắn trong menu **Tin nhắn**
2. Tạo công việc QC hoặc Phân loại
3. Chạy phân tích AI
4. Xem kết quả, xuất CSV/Excel
5. Xem Dashboard với dữ liệu

## Xóa dữ liệu demo

Khi đã trải nghiệm xong, xóa dữ liệu demo trước khi dùng thật:

1. Vào **Dashboard**
2. Bấm **Xóa dữ liệu demo**
3. Xác nhận xóa

::: warning Lưu ý
Xóa dữ liệu demo sẽ xóa toàn bộ cuộc hội thoại, tin nhắn và kết quả phân tích liên quan đến dữ liệu demo. Dữ liệu thật (nếu có) không bị ảnh hưởng.
:::

## Kịch bản trải nghiệm đề xuất

1. **Import demo** → Xem tin nhắn
2. **Cấu hình AI** (nếu chưa) → Nhập API key
3. **Tạo công việc QC** → Dùng mẫu quy tắc CSKH → Chạy thử
4. **Tạo công việc Phân loại** → Thêm nhãn: Khiếu nại, Góp ý, Hỏi giá, Đặt bàn → Chạy thử
5. **Xem kết quả** → So sánh 2 loại kết quả
6. **Xuất CSV/Excel** → Kiểm tra nội dung file
7. **Xóa demo** → Bắt đầu dùng thật
